## Yamada UI

https://yamada-ui.com/ja

Yamada UI は、日本人開発者によって作られた React 向けの UI コンポーネントライブラリ です。以下のような特徴があります。　　

主な特徴
1. 日本語ドキュメントが充実

　公式ドキュメントが日本語で書かれているため、英語が苦手な方でも学習しやすい　

　日本人開発者のコミュニティがあり、質問しやすい環境　

2. 豊富なコンポーネント  

　ボタン、フォーム、モーダル、テーブルなど、よく使うコンポーネントが揃っていてアクセシビリティにも配慮された設計  

3. カスタマイズ性が高い

　テーマのカスタマイズが容易
　デザインシステムとして活用できる　

4. TypeScript 対応

　型安全性が確保されている
　開発体験が向上

 5. 柔軟なスタイリングシステム  

  Yamada UIは独自のスタイリングシステムを採用しており、プロップスを通じて直感的にスタイルを適用できます。また、テーマのカスタマイズも容易で、プロジェクト全体の一貫したデザインを維持しやすくなっています。

6.充実した日本語ドキュメント
  日本語での詳細なドキュメントが提供されており、日本人開発者にとって導入・学習しやすいライブラリとなっています。

### DaisyUIとの違い
- DaisyUIはCSSクラス　HTMLだけで使える
- Yamada UIはReactコンポーネント React前提

### shadcn/uiとの違い
```bash
npx shadcn@latest add button
```
を実行すると自分のプロジェクト内にButtonコンポーネントのコードが生成されます。　

つまり、<Button>を使ってるけど実際にはcomponents/ui/button.tsxを使っているので
```TypeScript
const Button = (...)
```
の中身を好きなように編集て、自分で変更する前提です。
