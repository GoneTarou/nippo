## Scopeとは　　
https://railsguides.jp/active_record_querying.html

Railsのスコープ（scope）とは、「よく使う検索条件に名前を付けて再利用できる機能」 

例えば、日記アプリで「公開されている日記」だけを何度も取得したいとします。
```ruby
Diary.where(published: true)
```
毎回こう書くと大変なのでスコープを定義します。
```ruby
class Diary < ApplicationRecord
  scope :published, -> { where(published: true) }
end
```
```ruby
Diary.published
```
だけで同じ結果になります。
### スコープを使う場合
```ruby
class Diary < ApplicationRecord
  scope :published, -> { where(published: true) }
end
```
```ruby
Diary.published
Diary.published.order(created_at: :desc)
Diary.published.limit(10)
```
「公開されている日記を新しい順に10件取得する」
### 引数を受け取るスコープ
例えばユーザーごとの日記を取得したい場合
```ruby
class Diary < ApplicationRecord
  scope :by_user, ->(user) { where(user: user) }
end
```
使い方
```ruby
Diary.by_user(current_user)
```
### スコープはつなげられる
```ruby
class Diary < ApplicationRecord
  scope :published, -> { where(published: true) }
  scope :recent, -> { order(created_at: :desc) }
end
```
```ruby
Diary.published.recent
```
### スコープとクラスメソッドの違い
```ruby
scope :published, -> { where(published: true) }
```
```ruby
def self.published
  where(published: true)
end
```
どちらも
```ruby
Diary.published
```
で呼び出せる。　

一般的には、単純な検索条件 → scope　
複雑な処理や分岐がある → クラスメソッドという使い分けがよくされます。
