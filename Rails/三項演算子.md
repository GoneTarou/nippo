## 三項演算子
三項演算子（ternary operator）とは、条件によって2つの値のどちらかを返すための短い書き方です。
```ruby
条件 ? 真の場合 : 偽の場合
```

```ruby
if age >= 18
  puts "大人"
else
  puts "子供"
end
```
これを三項演算子で書くと
```ruby
puts age >= 18 ? "大人" : "子供"
```

### ログイン状態
```ruby
message = logged_in? ? "ログイン中" : "未ログイン"
```
```ruby
message
#=> "ログイン中"
```
ログインしてるなら"ログイン中" してないなら"未ログイン"

### 三項演算子を使うべき場面
「条件によって値を1つ選ぶだけ」なら、三項演算子は読みやすく便利です。

### 三項演算子を使わない方がよい場面
条件ごとに複数の処理をしたい場合は、if の方が適しています。
```ruby
if user.admin?
  logger.info "管理者がログイン"
  redirect_to admin_dashboard_path
else
  logger.info "一般ユーザーがログイン"
  redirect_to root_path
end
```
このように複数行の処理がある場合は、三項演算子より if の方が読みやすく、Railsでも一般的
